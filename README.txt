DOT. LASER - MVP Supabase + Vercel

1) No Supabase:
- Acesse seu projeto
- Vá em SQL Editor
- Cole o conteúdo de schema.sql
- Clique em Run

2) Teste local:
- Abra index.html no navegador
- Faça um cadastro
- Abra painel.html no navegador
- Busque o pedido

3) Publicação na Vercel:
- Crie um repositório no GitHub com estes arquivos OU use Vercel CLI
- Publique como projeto estático
- Página pública: /index.html
- Painel interno: /painel.html

IMPORTANTE:
Este MVP usa políticas abertas para facilitar o teste inicial. Antes de usar em evento real, devemos adicionar login de operador e regras mais restritas no Supabase.
